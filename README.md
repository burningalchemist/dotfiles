# dotfiles

# Init

```shell
# Install Homebrew packages
brew bundle install

# Copy necessary dotfiles

# Initialize Neovim. Lazy Package Manager will set up all the plugins.
# I'm on the same page[1] with @fatih when it comes to setting up vim/nvim.
nvim
```
